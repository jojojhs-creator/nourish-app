# Coffee, Cats & Malak Content Calendar

Tracks the scenario calendar used by `/coffee-cats-malak-daily`
(see `docs/COFFEE_CATS_MALAK_RUNBOOK.md`). Mark a scenario `Done` once
`/coffee-cats-malak-daily` has generated and posted it.

Supersedes `docs/StoryOwl_60_Day_Plan.md` (retired — old bedtime-story format).

## Format

| Field | Notes |
|---|---|
| Length | `15s` = single clip. `30s` = two clips, sequential beats of one scene. |
| Cast | Which of Malak / Olive / Mocha / Sky appear. |
