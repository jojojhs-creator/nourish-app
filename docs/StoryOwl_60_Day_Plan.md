# StoryOwl 60-Day Content Plan

Tracks the episode calendar used by `/storyowl-daily`
(see `docs/STORYOWL_RUNBOOK.md`). Languages alternate EN/AR. Mark an episode `Done`
once `/storyowl-daily` has generated and posted it.

## Content types

| Code | Type | Notes |
|---|---|---|
| Bedtime | Bedtime story | Calming; Clip 4 includes "goodnight" / candle-out outro |
| Fable | Life-lesson fable | Animal characters, clear moral |
| Educational | Educational short | "Did you know" hook + story that illustrates the fact |
| Adventure | Adventure micro-story | Hero faces one challenge; uplifting resolution |
| Funny | Funny kids story | Light comedy; relatable, shareable |
| Folklore | Cultural folklore | Traditional story retold in 60 seconds |
