StoryOwl — Privacy Policy

Last updated: June 14, 2026

StoryOwl is a personal content project that produces short bedtime-story videos and posts them to YouTube and TikTok. This page describes how the StoryOwl automation handles data when connecting to TikTok's API.

What StoryOwl accesses

StoryOwl uses TikTok's Content Posting API, authorized via OAuth, solely to upload and publish video content to the StoryOwl creator's own TikTok account.

  • Access tokens and refresh tokens used to authenticate API requests
  • Basic account/profile information required by the OAuth flow
  • Video files and captions generated by StoryOwl for posting

What StoryOwl does not do

  • StoryOwl does not collect data from TikTok viewers or other users
  • StoryOwl does not sell, share, or transfer any data to third parties
  • StoryOwl does not use TikTok data for advertising or analytics

Data storage

OAuth credentials are stored as encrypted secrets in a private GitHub repository and are only used by an automated posting workflow to publish StoryOwl's own videos.
